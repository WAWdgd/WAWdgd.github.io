如何打造一款爆款博客_02.简单配置并更换一个主题

发表于 2019-07-04 | 分类于 前端
字数统计: 679 | 阅读时长 ≈ 2

简单的配置

你的博客现在已经初步搭建好了，让我们来做更细致的调整吧。

现在回到命令行，按下Ctrl+C关闭你的博客。

为了让你的博客看起来更像是你的，打开你安装博客的文件夹，找到站点的_config.yml配置文件(以后安装了主题还会出现主题的配置文件，注意不要混淆)，用记事本打开。目前我们需要更改这几个地方：

title:网站主标题，会显示在浏览器标签栏等处

subtitle: 网站副标题

description: 个人介绍

author: 你的名字

language: zh-Hans(这里为更改博客语言，需要在更换主题后确定，next主题默认简体中文为此，目前就这样即可)

其他参数后面没有内容的，需要用一个空格填上以免报错。保存时请选择文件-另存为，将下方编码改为UTF-8，覆盖保存即可。最终效果图如下：

更换主题

你可能不太喜欢选择现在的默认主题，我们来安装一个新主题吧！

Next主题也许是目前hexo博客中最知名的博客主题(Next主题的官网是https://theme-next.iissnan.com/你可以在里面找到更多实用的帮助文档)，接下来的教程我们将会使用Next主题进行演示。当然，你也可以选择你喜欢的主题，不过和这个教程出入可能会比较大。(知乎大佬整理的好看的hexo主题)

现在开始安装Next主题。打开命令行，输入:

git clone https://github.com/iissnan/hexo-theme-next themes/next

完成后，Next主题就已经下载到博客目录的themes文件夹下的next文件夹中了。打开站点的_config.yml配置文件，修改主题为”next”：

现在，你的博客主题就安装完成了，你可以输入：

hexo g
hexo s

预览修改后的效果。

将网站关闭后，再次回到你的博客的文件夹，进入themes-next，打开主题的配置文件_config.yml：

按下Ctrl+F，搜索Schmes，找到如图所示的文本：

Next主题内置了Muse、Mist、Pisces、Gemini四个主题样式，你可以删掉某个样式前的”#”号更改，例如我这里选择了scheme:Pisces，你可以一个一个尝试寻找你喜欢的主题，但不可以同时启动多个样式。

保存时请选择文件-另存为，将下方编码改为UTF-8，覆盖保存即可。

现在你完成了对你的博客最基础的设置，下一篇我们接着讲，如何进一步配置你的博客。
